Filter method

The filter() method of Array instances creates a shallow copy of a portion of a given array, filtered down to just the elements from the given array that pass the test implemented by the provided function.

The callbackFn have to

Syntax: filter(callbackFn)

Syntax: filter(callbackFn, thisArg)

Return value:A shallow copy of the given array containing just the elements that pass the test. If no elements pass the test, an empty array is returned.

The callback function must be used to check conditionals, that's the difference between other array methods seen before.

  • forEach(): implement callbackFn to each element in an array. But is not appropriate if you need to use condicionals
  • map(): implement callbackFn AND return a new array with the values generated by the callbackFn
  • filter(): implements callbackFn, and it needs to check the conditionals. The method returns a new array, but just with the values that match the conditionals, in other words, the Filter() method only uses the truthy values.
